## Description

## Requirements
* [Node.js 3.8.2 or higher](https:///)


## Setup
``` bash
# Install dependencies
$ npm install

# Serve with hot reload
$ npm run start

# Build for production and launch server
$ npm run build
$ npm run start

# Generate static project
$ npm run generate
```
